# Hackathon Project Generator

## Overview
This Claude skill helps you brainstorm, plan, and execute winning hackathon projects by providing:
- Project idea validation
- Technology stack recommendations
- MVP feature prioritization
- Implementation roadmaps
- Pitch templates

## How to Use

### Command 1: Generate Project Ideas
Ask: "Generate 5 hackathon project ideas for a 24-hour AI hackathon"

Claude will:
1. Create unique, feasible project ideas
2. Assess difficulty and viability
3. Suggest tech stacks
4. Estimate implementation time

### Command 2: Validate Your Idea
Ask: "I want to build a meal planning app using AI. Is this hackathon-ready? What's the MVP?"

Claude will:
1. Assess scope and feasibility
2. Define core features
3. Identify tech stack
4. Create implementation timeline
5. Highlight potential pitfalls

### Command 3: Create Implementation Roadmap
Ask: "Create a 24-hour implementation roadmap for a ChatGPT-based productivity app"

Claude will:
1. Break down into 4-6 hour blocks
2. Assign tasks (frontend, backend, design)
3. Set milestones and checkpoints
4. Identify critical path items
5. Flag potential blockers

### Command 4: Generate Pitch Template
Ask: "Create a winning pitch for my AI weather dashboard"

Claude will:
1. Structure the pitch (Problem → Solution → Demo → Impact)
2. Write compelling copy
3. Add presentation tips
4. Create timing guide
